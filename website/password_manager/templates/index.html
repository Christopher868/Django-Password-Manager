About Us

I built this website as a personal project to create a safer and more reliable way to store passwords, instead of keeping sensitive information in plain text notes or other insecure places. Any account information saved in the password manager is protected using a zero-knowledge encryption model, meaning the data is encrypted on the client side before itâ€™s ever stored. This ensures that only the user can access their information.

Because of how the system is designed, not even I, as the developer, can view or recover stored passwords. The focus of this project was to prioritize user privacy while still keeping the application easy to use. It also gave me hands-on experience working with encryption, secure data handling, and building privacy-focused features in a real-world application.
